# gyaru-moji-generator

This is a generator that can convert hiragana to gyaru-moji.

## Install

```shell
% npm install gyaru-moji-generator
```

## Usage

```shell
% gyaru-moji-generator
```
or
```shell
% npx gyaru-moji-generator
```
